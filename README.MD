VUE 中数据绑定过程 DEMO

从第5步开始，代码并不能运行在浏览器中，为了方便测试，直接在 node 环境中执行。

建议安装 v8.9.0 以上的 node 版本，低版本的 node 不能使用 ES6 的模块系统。

执行方式，进入对应的文件夹，比如想执行 step5 中的测试文件。

```
cd demo/step5
node --experimental-modules test.mjs
```

就能得出对应结果
